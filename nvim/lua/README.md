## Directory structure
```
.
├── global_settings.lua # Some global 
├── helper              # Includes some helper functions which I may use in my
                        # plugin-settings other various stuff
├── lazy_settings.lua   # the settings for the lazy-plugin manager by folke
├── luasnip_snippets    # my snippets for luasnip
├── old                 # my old lua-stuff which I don't use anymore
├── plugins             # my neovim-plugins, which I use
└── plugins.lua         # the plugin file which holds plugins without any
                        # settings
```
